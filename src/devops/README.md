# CornField 环境变量

CornField 环境变量存储在 `/cornfield/env` 内，格式使用 `<variable>=value` 。

| 变量         | 值         | 说明                       |
| ------------ | ---------- | -------------------------- |
| MONGODB_IP   | 172.18.0.1 | MongoDB 服务器的 IP 地址。 |
| REDIS_IP     | 172.18.0.1 | Redis 服务器的 IP 地址。   |
| MONGODB_PASS | password   | MongoDB 服务器的密码。     |
